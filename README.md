# Shopping App

This is a Kotlin Shopping List app with features like authentication, database, navigation, etc.

Features of the App - 
1) Authentication (Firebase Email Auth)<br>
      -> Create account<br>
      -> Sign in<br>
      -> Sign out<br>
      -> Delete account<br>
3) Database
      -> Firebase Realtime Database<br>
      -> Add shopping items<br>
      -> Edit shopping item<br>
      -> Delete shopping items<br>
      -> Real-time data updates<br>
4) Navigation
5) Sidebar / Drawer for easy access
6) AlertDialog boxes for confirmations
7) Dark mode and light mode compatibility
8) Share GitHub profile ID directly from the app
9) Clean and user-friendly interface

Technologies Used -<br>
      -> Kotlin<br>
      -> Firebase Authentication<br>
      -> Firebase Realtime Database<br>
